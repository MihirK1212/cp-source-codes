# Use of vectors instead of array

```cpp
vector<int> v(n)
vector<int> v(n,1) ...initialzie all items as 1
vector<bool> v(n,true) ...initialize all items to true

v.front()  ...returns first element of vector i.e. v[0]
v.back()  ...returns last element of vector i.e. v[n-1]

v.pop_back() ...removes the last element of vector v[n-1]...does not return any value
```
